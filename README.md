# 🌦 Weather App with User Authentication

A modern **React + TypeScript weather application** that allows users to search for weather in any city and **save their favorite cities for quick access**. The app uses **Firebase Authentication and Firestore** to manage users and store saved cities.

---

## 🚀 Features

* 🔐 **User Authentication**

  * Signup with email and password
  * Login / Logout using Firebase Authentication

* 🌍 **Weather Search**

  * Search weather for any city
  * Real-time data from **OpenWeather API**

* 💾 **Save Favorite Cities**

  * Logged-in users can save cities
  * Cities are stored in **Firebase Firestore**

* ⚡ **Quick Access Weather**

  * Saved cities appear in a list
  * Click **Show Weather** to instantly view weather

* 🧠 **User-based Data**

  * Each user only sees **their saved cities**

---

## 🛠 Tech Stack

**Frontend**

* React

**Backend / Services**

**API**
